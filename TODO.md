# compilation simplifié
La compilation C demandé :
```sh
gcc -Wall -Wextra -std=c99 -o FILE FILE.c
```

# compression simplifié
Commandes déjà utilisées :
```sh
tar -czvf nom_archive.tar.gz FICHIER...
```

Décompression
```sh
tar -zxvf nom_archive.tar.gz
```
